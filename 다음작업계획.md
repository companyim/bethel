## 다음에 할 작업

### 1. 학년별 출석체크와 부서별 출석체크 분리
- 현재: 출석 체크에서 학년 필터와 부서 필터를 함께 사용 가능
- 변경: 두 개의 별도 섹션으로 분리
  - **학년별 출석체크**: 학년만으로 필터링하는 별도 섹션
  - **부서별 출석체크**: 부서만으로 필터링하는 별도 섹션
- 구현 방안:
  - 출석 체크 컴포넌트를 두 개로 분리하거나
  - 탭/섹션으로 구분하여 표시

### 2. 학생 추가 시 부서까지 한번에 입력
- 현재: 학생 추가 폼에 부서 선택 필드가 있음
- 개선 사항:
  - 학생 목록에 부서 정보 표시 (이미 완료됨)
  - 부서 선택 UI 개선 (필수/선택 명시)
  - 학생 추가 시 부서 정보가 함께 저장되는지 확인

### 3. 부서 출석 시에도 1달란트 지급
- 현재: 출석 체크에서 부서 필터 사용 시에도 달란트 지급 로직이 작동함
- 확인 사항:
  - 부서별 출석 체크가 별도로 분리되면 동일한 달란트 지급 로직이 필요함
  - 학년별 출석 체크와 동일한 달란트 지급 로직 적용
