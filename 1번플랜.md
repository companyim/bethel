## 주요 기능

### 1. 학생 관리 (관리자 전용)
- Excel 파일 불러오기 (A: 번호, B: 학년, C: 이름, D: 세례명)
- 수동 학생 추가/삭제
- 학년: 1, 2, 4, 5, 6, 첫영성체, 유치부

### 2. 출석 체크 (관리자 전용)
- **날짜 선택**: 2026년 일요일 목록만 표시
  - 드롭다운 또는 버튼 리스트 형태
  - 2026년의 모든 일요일 날짜 계산 및 표시
  - 날짜 형식: "2026.01.05 (일)" 형태
  - 선택된 날짜 하이라이트
- **학년 필터 (필수)**: 학년을 선택해야만 학생 목록 표시
  - 학년 미선택 시: "학년을 선택해주세요" 메시지
  - 학년 선택 시: 해당 학년 학생 목록만 표시
- 출석/결석 버튼 (지각 제거)
- 출석 저장 기능
- **달란트 지급 시스템**
  - 출석 저장 시 출석한 학생에게 자동으로 1달란트 지급
  - 결석한 학생은 달란트 미지급

### 3. 달란트 표시
- 학생 이름 뒤에 달란트 표시
  - 형식: "홍길동 (세례명) [달란트: 5]"
  - 학생 목록, 출석 체크 목록, 출석 기록 조회 등 모든 곳에 표시

### 4. 총 지급된 달란트 항목
- 관리자 대시보드에 추가
- 전체 학생에게 지급된 총 달란트 합계 표시
- 카드 형태로 표시

### 5. 출석 기록 조회
- **학생 모드**: 이름 + 학년 입력으로 조회
  - 학생의 총 달란트도 함께 표시
- **관리자 모드**: 전체 기록 조회 및 필터링

### 6. 관리자 모드
- 비밀번호 로그인
- 통계 대시보드
  - 전체 학생 수
  - 전체 출석 기록 수
  - 출석일 수
  - 출석/결석 통계
  - **총 지급된 달란트** (새로 추가)
- 데이터 내보내기/가져오기 (JSON, Excel)

## 데이터 구조

### 학생 데이터
```javascript
{
  name: string,
  grade: number | string, // 1-6 또는 "첫영성체", "유치부"
  baptismName: string,
  talent: number // 달란트 (기본값: 0)
}
```

### 출석 기록
```javascript
{
  date: string, // YYYY-MM-DD (2026년 일요일만)
  student: string,
  grade: number | string,
  status: 'present' | 'absent',
  talentEarned: boolean // 달란트 지급 여부 (출석 시 true)
}
```

## 유틸리티 함수

### dateUtils.js
- 2026년의 모든 일요일 날짜 배열 반환 함수
- 날짜 포맷팅 함수 (YYYY.MM.DD (일) 형식)
- 날짜 비교 함수

### talentUtils.js (새로 추가)
- 출석 시 달란트 지급 함수
- 총 달란트 계산 함수
- 달란트 표시 포맷팅 함수

## 개발 순서

1. Vite로 React 프로젝트 초기화
2. Context API 설정 및 상태 관리 (달란트 필드 추가)
3. 유틸리티 함수 (스토리지, Excel 파서, 날짜 유틸, 달란트 유틸)
4. 기본 레이아웃 및 Header
5. 관리자 로그인
6. 학생 관리 컴포넌트 (달란트 표시 추가)
7. 출석 체크 컴포넌트 (2026년 일요일 목록 + 학년 필터 필수 + 달란트 지급 로직)
8. 출석 기록 조회 컴포넌트 (달란트 표시)
9. 관리자 대시보드 (총 지급된 달란트 항목 추가)
10. 스타일링
11. 빌드 및 배포 테스트
