# 베텔 주일학교 출석부

React와 Vite를 사용한 출석 관리 웹 애플리케이션입니다.

## 주요 기능

### 학생 모드
- **출석 기록 조회**: 이름과 학년을 입력하여 개인 출석 기록 조회
- 달란트 확인: 개인 달란트 현황 확인

### 관리자 모드
- **학생 관리**
  - Excel 파일 불러오기 (A: 번호, B: 학년, C: 이름, D: 세례명, E: 부서)
  - 학생 수동 추가/삭제
  - 학년별 학생 목록 관리 (1, 2, 4, 5, 6학년, 첫영성체, 유치부)
  - 부서별 필터링 (성가대, 복음퀴즈부, 율동부, 전례부)
  
- **출석 체크**
  - 2026년 일요일 날짜 선택
  - 학년별 필터링
  - 부서별 필터링
  - 학년과 부서 동시 필터링 가능
  - 출석/결석 체크
  - 출석 시 자동 달란트 지급 (1회당 1달란트)
  
- **관리자 대시보드**
  - 전체 통계 확인
  - 총 지급된 달란트 현황
  - 데이터 내보내기/가져오기 (JSON, Excel)
  - Excel 파일 불러오기 (데이터 관리 섹션)
  - 모든 데이터 삭제 (비밀번호 확인 필요)
  
- **출석 기록 관리**
  - 학년/부서/학생/날짜별 필터링
  - 전체 출석 기록 조회
  - 부서 정보 표시

## 달란트 시스템
- 학생이 출석하면 자동으로 1달란트 지급
- 달란트는 학생 이름 옆에 표시됨 (예: 홍길동 (세례명) [달란트: 5])
- 관리자 대시보드에서 총 지급된 달란트 확인 가능

## 기술 스택
- React 18
- Vite
- xlsx (Excel 파일 처리)
- LocalStorage (데이터 저장)

## 설치 및 실행

### 1. 의존성 설치
```bash
npm install
```

### 2. 개발 서버 실행
```bash
npm run dev
```

### 3. 빌드
```bash
npm run build
```

빌드된 파일은 `dist` 폴더에 생성됩니다.

### 4. 배포
`dist` 폴더의 내용을 웹 호스팅 서비스에 업로드하면 됩니다.

## 데이터 저장
- 모든 데이터는 브라우저의 LocalStorage에 저장됩니다.
- 관리자 비밀번호도 LocalStorage에 저장됩니다.
- 데이터 백업을 위해 관리자 대시보드에서 JSON/Excel 내보내기를 권장합니다.

## Excel 파일 형식
- A열: 번호 (무시됨)
- B열: 학년 (1, 2, 4, 5, 6 또는 "첫영성체", "유치부")
- C열: 이름
- D열: 세례명 (선택사항)
- E열: 부서 (성가대, 복음퀴즈부, 율동부, 전례부)

## 부서 시스템
- 학생별 부서 지정 가능
- 부서 목록: 성가대, 복음퀴즈부, 율동부, 전례부
- 출석 체크 시 부서별 필터링 가능
- 출석 기록에 부서 정보 저장

## 주의사항
- 관리자 비밀번호는 최초 로그인 시 설정됩니다.
- 데이터는 브라우저별로 저장되므로 다른 브라우저에서는 데이터가 공유되지 않습니다.
- 브라우저 데이터를 삭제하면 모든 데이터가 삭제됩니다.
